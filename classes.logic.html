classes.logic.deck module¶

class classes.logic.deck.Deck(qty=1)¶

Bases: object

A deck of cards and actions.

A standard deck consists of 52 Cards from 2 - A of K, H, D, and S suits. This class allows for multiple decks to be used alongside a marker card to signify shuffling.

Attributes:
_pips: (List): The list of pips to use. _suits: (List): The list of suits to use. _marker: (Card): The marker card used to signify reshuffle point.
